Política de Privacidade

Última atualização: 08 de agosto de 2025

Esta Política de Privacidade explica como coletamos, usamos e protegemos os dados pessoais dos usuários do nosso aplicativo.

1. Dados que coletamos

  • Informações que você fornece diretamente (ex.: nome, e-mail).
  • Dados coletados automaticamente (ex.: uso do app, IP, dispositivo).

2. Uso dos dados

Utilizamos os dados para melhorar a experiência do usuário, oferecer suporte e enviar atualizações importantes.

3. Compartilhamento de dados

Não vendemos seus dados a terceiros. Podemos compartilhar dados com fornecedores confiáveis para funcionamento do app.

4. Segurança

Tomamos medidas para proteger seus dados contra acesso não autorizado.

5. Direitos do usuário

Você pode solicitar acesso, correção ou exclusão dos seus dados pessoais.

6. Contato

Para dúvidas sobre esta política, entre em contato: email@exemplo.com
